Assignment 2 has the following files:


Client.java:creates request for the server
IntermediateHost.java:Helps the client and  server communicate with eachother
Server.java: Holds the server that responds to client requests


To run the program:


1. Extract the SYSC3303_ASSIGNMENT2.zip
2. Open the SYSC3303_ASSIGNMENT2.zip folder
3. Copy the source code for the file and put it in eclipse 
3. Run the main method in Server.java
4. Run the main method in IntermediateHost.java
5. Run the main method in Client.java


Note: This project was developed in intelli j so you would have to copy the code from the zip file to run it
